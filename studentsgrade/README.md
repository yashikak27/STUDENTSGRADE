# Task 2 - Student Grade Calculator

## Description

This Java application calculates the total marks, average percentage, and grade of a student based on marks entered for different subjects.

## Features

* Accepts marks for multiple subjects
* Calculates total marks
* Calculates average percentage
* Displays grade based on percentage

## Grade Criteria

| Percentage   | Grade |
| ------------ | ----- |
| 90 and above | A     |
| 80 - 89      | B     |
| 70 - 79      | C     |
| 60 - 69      | D     |
| Below 60     | F     |

## Technologies Used

* Java
* Scanner Class

## How to Run


```bash
javac StudentGradeCalculator.java
```


## Learning Outcomes

* Loops
* Conditional Statements
* Arithmetic Operations
* User Input Handling
